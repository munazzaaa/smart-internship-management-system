# smart-internship-management-system
A database-driven system to manage internships, applications, and student-company interactions efficiently.

 📌 Project Overview

The Smart Internship Management System is a database-driven web platform designed to streamline and automate internship processes in educational institutions. It connects students, companies, and administrators in one centralized system.

> Problem Statement

Traditional internship management systems rely on manual processes such as emails and spreadsheets, which lead to:

* Data inconsistency
* Lack of transparency
* Poor communication
* Difficulty in tracking applications

> Proposed Solution

This system provides a centralized platform where:

* Students can apply for internships
* Companies can post opportunities
* Admin can manage and monitor the entire process

> Key Features

* User registration and login
* Student profile management
* Internship posting and search
* Application submission and tracking
* Admin approval system
* Feedback and evaluation system
* Notifications

>  Database Design

Entities:

* Students
* Companies
* Internships
* Applications
* Feedback

Relationships:

* One student → many applications
* One company → many internships
* One internship → many applications

> Technologies Used

* Frontend: HTML, CSS, JavaScript
* Backend: Flask / PHP
* Database: MySQL
